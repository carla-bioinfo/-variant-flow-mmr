# 🎓 MODELO DE APRENDIZAGEM ESTRUTURADA

## 4 PILARES

1. **Scaffolding Progressivo** - O QUÊ → POR QUÊ → COMO → ONDE → IMPLEMENTAÇÃO
2. **Execução + Análise** - Você executa, Claude analisa
3. **Validação Contínua** - Checklist antes de avançar
4. **Consolidação** - Resumo + commit + documentação

## RODADA PADRÃO (15 min)

**Minuto 0-1**: Explicação (O QUÊ/POR QUÊ/COMO)
**Minuto 1-2**: Apresentar 3 comandos/conceitos
**Minuto 2-3**: Você executa (✅ COMANDO 1)
**Minuto 3-5**: Análise + Checklist

## SESSÃO PADRÃO (45-60 min)

- 0-2 min: SETUP
- 2-5 min: CONTEXTO
- 5-30 min: 3-4 RODADAS
- 30-40 min: CONSOLIDAÇÃO
- 40-50 min: VALIDAÇÃO
- 50-60 min: PRÓXIMA SESSÃO

## CHECKLIST FINAL

□ Pode explicar sem notas?
□ Pode fazer na prática?
□ Pode ensinar outro?
□ Pode adaptar para novo contexto?
