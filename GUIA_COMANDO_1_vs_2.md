# 🎯 GUIA: COMANDO 1 vs COMANDO 2

## REGRA DE OURO
✅ COMANDO 1 = EXECUTE AGORA

├─ Copie o código

├─ Cole no terminal

├─ Aperte ENTER

└─ Mostre resultado aqui
❌ COMANDO 2 = LEIA APENAS

├─ Leia para entender

├─ NÃO cole

├─ NÃO execute

└─ É referência/exemplo
NÃO TEM RÓTULO? = PERGUNTE ANTES

"Isto é COMANDO 1 ou 2?"
## EXEMPLOS

### ✅ COMANDO 1: Verificar venv
```bash
echo "Venv: $VIRTUAL_ENV"
```
→ Você executa e mostra resultado

### ❌ COMANDO 2: Exemplo de classe
```python
class MinhaClasse:
    def __init__(self):
        self.valor = 10
```
→ Você lê, entende, NÃO executa

## QUANDO TER DÚVIDA

Se não tem certeza:
- Pergunta: "Isto é COMANDO 1 ou 2?"
- Espera resposta
- NÃO executa até confirmar

**Melhor perguntar do que quebrar!** 🎯
